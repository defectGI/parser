EKSIKLER
========
Hatırlatma listesi, dokümantasyon değil. Çözülünce satır silinir; yeni sorun
bulununca tek satır eklenir. Detay için parsers/README.md ve SCOPE.txt.

docx
----
- styles.xml'e gömülü bold/font boyutu mirası okunmuyor (başlık + inline
  formatting'i etkiler)
- Kapak/roster/isim listesi sayfaları format skoruyla gerçek başlıktan ayırt
  edilemiyor
- Başlık yanlış-pozitifi table_describe breadcrumb'ine sızabiliyor

pptx
----
- Tablo hücre metninde inline formatting (bold/italic) kayıp
- Native chart/SmartArt hiç yakalanmıyor (kapsam dışı kararı)

xlsx
----
- Hücre-içi zengin metin/format (runs) okunmuyor
- Formül kayboluyor, sadece son hesaplanmış değer kalıyor
- Sheet içindeki birden fazla mantıksal tablo tek TableBlock'a düzleşiyor
- Chart/pivot tablo desteklenmiyor (kapsam dışı kararı)
- Hücre-tabanlı başlık tespiti yok

html
----
- Tablo hücresinde inline formatting (bold/italic) modellenmiyor

markdown
--------
- Blockquote, reference-style link/görsel, inline HTML işlenmiyor
- İç içe/üst üste binen emphasis (bold içinde italic gibi) taranmıyor
- Çok satırlı setext başlık metni yakalanmıyor

pdf
---
- VLM bbox vermezse (tahmin veremediği durumlarda) tek-raster figürde crop
  yine sayfa geneli kalıyor
- Inline formatting (runs): sadece code path'te (font adından bold/italic);
  hybrid/scanned'de (VLM okuması) hâlâ yok — doğrulanacak bağımsız kaynak yok
- Başlık seviyesi sayfa-içi font sıralamasıyla belirleniyor (doküman geneli değil)
- Span'de byte offset yok, sadece sayfa numarası

Genel
-----
- Bozuk/0-byte zip dosyasında (docx/pptx/xlsx) ham exception dışarı sızıyor
